# 让 Chrome 更好用

## 1.插件推荐

- **Tampermonkey**

    油猴脚本，安装后自行搜索各种脚本添加至该插件，可以在「Greasy Fork」网站上找到很多脚本

- **Elasticvue**
 
    可视化管理 ElasticSearch 的插件

- **File Icons for GitHub and GitLab**
 
    在 GitHub 和 GitLab 上显示文件图标

- **JSON Viewer Pro**
  
    JSON 格式化插件

- **AdGuard**

    广告拦截插件

- **GitZip for github**

    无需 clone，即可下载 github 项目中单个或多个文件

- **Vue.js devtools**

    Vue.js 调试插件
